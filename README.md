# sensortracker

Backend di un'applicazione che traccia e salva su DB dati:
- Parametri vitali:
  - Peso
  - Altezza
  - BMI
  - BMR
  - Grasso Sottocutaneo
  - Grasso Viscerale
  - ...
- Prestazioni sul percorso di andata/ritorno da sensori posti a diverse distanze:
  - Tempi di reazione
  - Tempi di raggiungimento di ogni singolo sensore
